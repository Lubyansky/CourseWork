В качестве ядра приложения используется Node.js.
Реализуемая архитектура: REST API.

Шаблоны html документов реализованы с помощью ejs-шаблонизатора, файлы шаблонов находятся в папке templates.
Фронтэнд базируется на фреймворке Vue.js.
Все файлы отвечающие за фронтэнд расположены в /static/assets.

В качестве базы данных была использованна система PostgreSQL расположенная на локальном сервере, поэтому данные для статей были вынесены в виде массивов в отдельные файлы, 
но я настроил доступ к базе данных на моём ПК, так что можете попробовать зайти, вдруг компьютер будет включен :).
Для этого надо открыть файлы в папке controllers, закомментировать текущий код, и снять коментарии с заготовленного кода (пометил в файле). 
Конфигурация для подключения уже готова и вынесена в отдельный файл.

Для запуска приложения требуется ввести в терминале команду: 

npm start — стандартный запуск;
или 
npm run dev — запуск через утилиту nodemon (автоматический перезапуск сервера при обновлении файлов)).
