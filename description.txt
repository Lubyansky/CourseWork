В качестве ядра приложения используется Node.js.
Реализуемая архитектура: REST API.

Шаблоны html документов реализованы с помощью ejs-шаблонизатора, файлы шаблонов находятся в папке templates.
Фронтэнд базируется на фреймворке Vue.js.
Все файлы отвечающие за фронтэнд расположены в /static/assets.

Для запуска приложения требуется ввести в терминале команду: 

npm start — стандартный запуск;
или 
npm run dev — запуск через утилиту nodemon (автоматический перезапуск сервера при обновлении файлов)).
